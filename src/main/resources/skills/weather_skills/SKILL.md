## 功能概述
此技能用于查询全球主要城市的实时天气状况，包括温度、天气描述等。  
**实际逻辑由后端 Spring Boot 应用中的 Java 方法实现**，Claude 仅负责触发调用并格式化结果。


## 使用规则
- 用户必须提供**城市名称**（支持中文或英文）。
- 如果用户未指定城市，请先询问：“请问您想查询哪个城市的天气？”
- 不支持模糊查询（如“附近天气”），必须明确城市。

## 输出格式
严格按以下 JSON 格式返回（即使在对话中也要保持结构）：
```json
{
  "city": "北京",
  "temperature_c": 25,
  "condition": "晴",
  "humidity_percent": 60
}
```
